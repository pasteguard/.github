# PasteGuard

PasteGuard is an early-stage project exploring ways to protect sensitive information during everyday clipboard usage.

The focus is on local-first mechanisms that help reduce accidental exposure of confidential data in development and technical workflows.

## Scope

PasteGuard is currently in a research and validation phase.  
The public codebase reflects exploratory work and prototypes used to evaluate detection, prevention, and user interaction concepts around clipboard safety.

## Repositories

This organization contains repositories related to the PasteGuard project.  
Each repository includes its own README with technical details and build instructions.

## Status

PasteGuard is under active development.
